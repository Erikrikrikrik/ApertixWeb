Sobre Nuestro Robot

Ñebot es un robot diseñado para participar en el concurso Apertix, con el objetivo de ganar la competición con la máxima puntuación posible. Su papel principal es coger flores y árboles para después colocarlos en sus respectivas posiciones. Ñebot cuenta con un sistema de navegación avanzado con el cual puede detectar colores y saber la ubicación exacta en la que se encuentra respecto a las posiciones. El diseño del robot cuenta con unas garras semejantes a las del escarabajo rey, de hecho, está inspirado en él.

Hardware y Componentes

Ñebot está construido con componentes de alta calidad, incluyendo motores de alto rendimiento, una placa de control avanzada y sensores de precisión para una navegación eficiente.

Programación

El robot está programado en bloques, utilizando el entorno de programación visual para facilitar el desarrollo y el entendimiento del código. A pesar de estar programado en bloques, cuenta con una lógica de control robusta para manejar las tareas complejas y con la que evitar errores.

El Mayor Reto

El mayor reto fué crear el diseño inicia, ya que tuvimos que tirar de toda nuestra creatividad, pero una vez nos inspiramos en el escarabajo rey, encontramos un diseño perfecto con el cual ganar la competición e integrar todas las caracteristicas esenciales.
